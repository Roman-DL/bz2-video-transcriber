# Инструкции по извлечению

Извлеки всю текстовую информацию с представленных слайдов.

## Порядок обработки

1. **Определи заголовок слайда** — обычно крупный текст вверху
2. **Извлеки основной контент** — текст, списки, данные
3. **Преобразуй таблицы** — в markdown формат с `|` разделителями
4. **Опиши визуальные элементы** — графики, схемы, диаграммы

## Особое внимание

- **Таблицы с данными**: сохрани все колонки и строки
- **Числовые показатели**: проценты, суммы, количества
- **Источники**: если указаны ссылки или авторы

## Формат ответа

Верни структурированный markdown без дополнительных комментариев.
Начинай сразу с контента слайдов.
