# Шаблон: Лонгрид обучающей темы

## Структура выходного JSON

```json
{
  "introduction": "Вступление (1-2 абзаца): контекст темы, почему важна",
  "sections": [
    {
      "title": "Название раздела по содержанию",
      "content": "Изложение от первого лица..."
    },
    {
      "title": "Следующий раздел",
      "content": "Каждый раздел должен быть самодостаточным для RAG..."
    }
  ],
  "conclusion": "Заключение (если спикер подводил итог)",
  "topic_area": ["область1", "область2"],
  "tags": ["тег1", "тег2", "тег3"]
}
```

## Правила заполнения

### introduction
- 1-2 абзаца
- Контекст темы: почему важна
- Если спикер представляется — включить кратко

### sections
- Разделы по логике автора
- Заголовки по содержанию (не «Раздел 1»)
- Каждый раздел самодостаточен
- Изложение от первого лица

### conclusion
- Итог, если спикер подводил
- Может быть пустым, если не было явного заключения

### topic_area
- 1-3 области из допустимого списка
- Выбирать по основному содержанию

### tags
- 3-7 тегов для поиска
- Конкретные термины, инструменты, концепции
