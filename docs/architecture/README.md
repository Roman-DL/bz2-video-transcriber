## Когда создавать документ

- Реализована новая подсистема (сервис, интеграция)
- Нетривиальный поток данных
- Модуль с собственными решениями и API

## Формат

Каждый документ: `NN-module-name.md`

Содержит: схемы, API, потоки данных, файловая структура, ключевые решения.

**Источники:** реальный код (первичный), requirement, plan.
