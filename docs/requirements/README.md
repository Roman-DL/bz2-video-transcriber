## Статусы

- **Draft** — обсуждение, уточнение
- **Ready** — согласовано, готово к планированию
- **In Progress** — план создан, идёт реализация
- **Done** → перемещается в `archive/`

## Как добавить

1. Обсудить фичу в Claude Desktop
2. Попросить сформировать документ по шаблону
3. Сохранить как `docs/requirements/{feature-name}.md`
4. Обновить эту таблицу
